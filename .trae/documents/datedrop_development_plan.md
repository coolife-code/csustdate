# CSUST DateDrop 项目开发计划

## 项目概述

**项目名称**: CSUST DateDrop - 长沙理工大学学生配对网站  
**开发周期**: 7-9周  
**技术栈**: 
- 前端: Vue 3 + Vite + Tailwind CSS + Pinia
- 后端: Node.js + Koa + SQLite + Sequelize
- 部署: PM2 + Nginx

**服务器配置**: 2核2G轻量级配置

## 技术要点

### 后端关键技术

1. **SQLite优化**
   - 启用WAL模式提高并发性能
   - 设置合理的缓存大小
   - 使用索引优化查询

2. **匹配算法**
   - 稳定室友算法实现
   - 特征向量相似度计算
   - 多轮匹配机制

3. **性能优化**
   - 连接池管理(最大5个连接)
   - 批量数据库操作
   - 向量预计算

### 前端关键技术

1. **性能优化**
   - 路由懒加载
   - 组件按需加载
   - 图片懒加载

2. **用户体验**
   - 表单自动保存
   - 加载状态提示
   - 错误边界处理

---

## 风险控制

### 技术风险
- **匹配算法复杂度**: 使用候选过滤降低计算量
- **内存限制**: 严格限制并发连接数和缓存大小
- **数据库性能**: 使用索引和查询优化

### 进度风险
- **功能优先级**: 核心功能优先,次要功能可延后
- **技术难点**: 提前研究匹配算法和向量计算
- **测试时间**: 预留充足的测试和修复时间
